**1. Implement a Python function that “inverts” a dictionary where keys are string, and values are integers.
So that dictionary {‘k1’: v1, ‘k2’: v2} would become {v1: ’k1’, v2: ‘k2’}

```python
for i in range(1, 101):
    if i % 3 == 0 and i % 5 == 0:
        print('Fizz Buzz')
    elif i % 3 == 0:
        print('Fizz')
    elif i % 5 == 0:
        print('Buzz')
    else:
        print(i)
```
